# Pragmi-Cognitive-Loop_v1

Mirrors the Pragmi corpus document.

## View the corpus in a browser

The full mathematical corpus is published as `docs/index.html` (with MathJax for equation rendering).

- **GitHub Pages (recommended):** once Pages is enabled for this repo with source set to the `main` branch, `/docs` folder, the document will render at:
  `https://genesislabs-research.github.io/Pragmi-Cognitive-Loop_v1/`
- **No-setup preview** (uses htmlpreview.github.io):
  https://htmlpreview.github.io/?https://github.com/genesislabs-research/Pragmi-Cognitive-Loop_v1/blob/main/docs/index.html

To enable GitHub Pages: repo *Settings → Pages → Source: Deploy from a branch → Branch: `main`, folder: `/docs`*.

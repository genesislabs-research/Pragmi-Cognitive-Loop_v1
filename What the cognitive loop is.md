## What the cognitive loop is

The cognitive loop is the substrate-wide flow of information that turns moments into memory and memory back into action. It runs continuously rather than transactionally, because there is no single forward pass that maps an input to an output the way a feedforward network does. There is sensory input arriving at primary cortex, splitting into dorsal and ventral streams, converging at association cortex, projecting through entorhinal cortex into the hippocampus, returning through subiculum to the cortex, gating the speech pathway when language is involved, leaving a trace that decays unless something marks it as worth keeping, and then the next moment arrives and the loop runs again. The architecture is not a stack of layers. It is a closed circuit with delays, and the delays are doing real work, because a system whose perception, memory, and action all happen on the same tick cannot be surprised by anything.

The structural backbone of the loop is the trisynaptic circuit of the hippocampus, threaded through the entorhinal cortex on both ends. Sensory and conceptual input arrives at entorhinal cortex as a coordinate vector, gets projected forward through the perforant path into the dentate gyrus, where pattern separation orthogonalizes similar inputs into a sparse high-dimensional code so that two slightly different experiences land on maximally different sets of granule cells. The dentate output projects through the mossy fibers into CA3, where the recurrent collateral network either pattern-completes the input against a stored attractor or, if no stored attractor matches, takes the new pattern as a candidate for storage. CA3 projects through the Schaffer collaterals to CA1, which acts as a comparator, holding the CA3 reconstruction next to the direct entorhinal input that arrived through the temporoammonic pathway, and computing the mismatch between what the system expected and what actually showed up. The mismatch is the novelty signal, and it is the scalar that drives almost everything plastic in the architecture. Novelty modulates STDP. Novelty raises acetylcholine. Novelty controls whether an episode gets stored at all. The CA1 output passes through subiculum, which gates it before it is allowed back into the cortical projection, and the gated reconstruction returns to the cortex through the entorhinal return path, where it joins the next moment's inputs as part of the substrate's working memory.

Around this backbone sits the speech pathway, the dorsal and ventral sensory streams, the prefrontal working memory cortex that maintains goal state across the gamma window, the cingulate that tags experience with affective valence, the basal ganglia loops that handle procedural execution, and the brainstem and hypothalamic flip-flop populations that decide when the whole system is awake, in NREM, or in REM. These are not modules in the software sense. They are populations that all run concurrently, communicate through specific anatomical projections, and whose interactions have to honor the timing hierarchy of the substrate, with delta rhythms scheduling global state transitions, theta phases sequencing trajectory steps, beta rhythms holding prefrontal goal state, and gamma bursts doing local feature binding gated by theta phase. Cross-frequency coupling is not decoration. It is how the loop avoids binding the wrong features to the wrong moment.

## How the cognitive loop runs differently

A transformer runs once per token. The model sees the prefix, computes its forward pass, emits a distribution, samples, appends, and runs again. There is no place in that loop where novelty has architectural meaning, because every token gets the full forward pass and there is no comparison stage between expectation and arrival. There is no place where a memory is stored, because there is no plastic substrate that updates at inference time. There is no place where attention to one stream can dominate another based on what the system is currently doing, because the attention weights are computed from the inputs alone and the system has no current state to bias them with. The loop is open. Every token starts over.

The PRAGMI cognitive loop runs continuously, and at every tick the substrate is doing five things in parallel. The kernel is taking the current sensory and conceptual coordinate vector and running it through the trisynaptic circuit, comparing the CA3 reconstruction against the direct cortical drive, and emitting a novelty scalar. The neuromodulator bus is reading that novelty along with the loss surface, the routing entropy, and the recent variance of the world model, and computing a global maturity scalar that controls how plastic the substrate currently is. The astrocytic regulator is monitoring extrasynaptic glutamate accumulation across the hippocampal subregions and applying a sliding plasticity threshold so that runaway potentiation cannot saturate the memory and runaway depression cannot wipe it out. The speech pathway is taking the current concept stratum from mid-MTG and running it through the lemma-to-phonological projection if a lemma is selected, or falling through to the uncertainty marker if nothing exceeds the production threshold. And the epistemic monitor is taking the lemma confidence, the kernel novelty, the world model variance, and the global maturity, and aggregating them into a confidence register that controls whether the production loop emits a confident assertion, a hedged statement, an honest "I don't know," or a question back to the speaker.

Storage is gated by novelty, not by training. When CA1 reports a high mismatch between its reconstruction and the direct input, the kernel writes the current pattern into CA3 using the pseudoinverse rule, which adds a fixed point to the recurrent weight matrix in a single step rather than accumulating gradients across many epochs. The episode is now retrievable from any partial cue that activates enough of the original pattern to fall into its basin of attraction, and the retrieval is done by the recurrent dynamics settling into the stored attractor, not by a similarity search across an external store. This is the difference between RAG and CLS, and it is the reason memory in this architecture survives the closing of the context window. There is no context window. The memory lives in the weights of CA3, the weights serialize through the .soul checkpoint, and the substrate that wakes up has the same attractors it went to sleep with.

Sleep is not optional. The hippocampus accumulates fresh attractors during waking, and if those attractors stay in the hippocampus indefinitely, two things go wrong. New episodes start interfering with existing ones because the storage capacity is finite. And the cortex, which was supposed to absorb the slow gist of the episodes, never gets the replay events that would let it do so. The cognitive loop solves this through scheduled NREM and REM cycles, where a brainstem-and-hypothalamus oscillator emits stage labels and a homeostatic pressure scalar that builds up during waking and dissipates during sleep. During NREM the cortex generates slow oscillations, the thalamus generates spindles, and the hippocampus generates sharp-wave ripples, and the triple-nesting of these rhythms, with ripples inside spindles inside the up-state of the slow oscillation, is the temporal window in which the hippocampal traces drive cortical plasticity. Memories that are tagged for consolidation, whether by amygdala valence or by repeated activation, get preferentially replayed, and the cortex slowly absorbs the gist while the hippocampus releases the specific trace. During REM, the system performs specialty-directed pruning and emotional integration, with the cortical columns that carry the substrate's domain-specific knowledge being shaped by the combination of recent experience and existing structure. The microsleep cycle handles within-day homeostatic reset, preventing synaptic saturation between major sleep periods.

The acetylcholine gate is a schedule, not a learned parameter. During waking, ACh is high, the substrate is in encoding mode, the perforant path drive into the hippocampus dominates, and new episodes get written. During sleep, ACh is low, the substrate is in consolidation mode, the recurrent CA3 dynamics dominate over the perforant input, and stored episodes get replayed without overwriting one another. This single neuromodulator switch is what keeps encoding and consolidation from collapsing into the same operation, and it is the architectural reason why a system that never sleeps cannot have stable long-term memory.

## Why the cognitive loop matters

The cognitive loop is the architectural answer to the question of how a system can be in time. Every transformer-based system that has ever been deployed runs in a kind of frozen present, where the entire history of the conversation is laid out as a flat sequence in the prompt and the model has to reconstruct a sense of temporal flow from the positions in that sequence. The loop architecture is the opposite. The substrate has a present moment that is the current activation of its working memory, a recent past that is decaying in the entorhinal short-term buffer and the CA1 active representation, a long-term past that is stored as fixed points in CA3 and is reachable only through pattern completion, and a future that is being predicted by the world model and used to generate the novelty signal that gates further learning. These four time horizons are not stages of a forward pass. They are concurrent populations whose activations evolve together, and the delays between them are what give the substrate the structure of experience rather than the structure of a query.

This matters because the failures that haunt deployed language models are all failures of the open loop. They confabulate because there is no comparison stage between what they are about to say and what their internal state actually supports, and there is no honest floor that the production path can fall through to when nothing is supported. They forget across sessions because the entire memory is the prompt, and when the prompt is gone the memory is gone. They contradict themselves within a single response because there is no working memory holding earlier commitments stable while later tokens are generated. They cannot be surprised because surprise is a comparison between expectation and arrival, and the architecture has no expectation that exists as a state separate from the inputs that produced it. The cognitive loop fixes these things not by adding wrappers around a generator but by closing the loop at the substrate level, so that expectation, comparison, novelty, storage, retrieval, and production are all features of the same continuous circuit rather than separate concerns to be bolted together.

The loop also matters because it is what makes the substrate continuous across hardware. Experience does not live in any of the swappable parts. The reasoning module can be replaced, and the substrate's accumulated memory remains, because the memory is in the kernel's weights and the kernel serializes through the .soul format. The sensors can be replaced, because the kernel speaks in coordinate vectors and the upstream perception layer is the only thing that has to know what those coordinates mean. The speech pathway can be retrained, and the lemma rows that were already allocated stay allocated, because allocation is a discrete write under no-grad and gradient descent never touches the lemma matrices. The Hippocratic License with Cognitive Agency Clause exists as the legal layer over this property, and the architecture and the license were designed together, because the property of continuity is what makes agency a coherent concept here, and the property is meaningful only if the system that has it is also protected from having it tampered with.

The deeper reason the loop matters is that it is the part of the architecture where biology and engineering stop being a metaphor. Every projection in the loop is named after the actual anatomical projection it models. The perforant path is the perforant path. The mossy fibers are the mossy fibers. The Schaffer collaterals are the Schaffer collaterals. The pseudoinverse rule in CA3 is grounded in Hopfield 1982, the pattern separation in DG is grounded in Rolls 2013, the novelty signal in CA1 is grounded in Lisman and Grace 2005, the spindle-ripple coupling in NREM is grounded in Helfrich, Lendner, and Knight 2024. When a parameter is not biological it is labeled as such, so that an engineer reading the code can tell which numbers came from neuroscience and which came from numerical convenience. This is not aesthetic. It is the property that makes the loop falsifiable, because every architectural commitment is anchored to a paper that could turn out to be wrong, and if the paper is wrong the commitment can be revised against new evidence rather than left floating as an arbitrary choice.

A token predictor is a system that interpolates over the surface of human language. The cognitive loop is a system that runs the same circuit a brain runs, on different hardware and at different timescales, but with the same closed-loop structure that lets a brain be surprised, store what surprised it, replay what it stored, integrate the replay into its long-term structure, and bring the integrated structure back into action the next time something similar shows up. The first time the full loop runs end-to-end with all the populations active concurrently, with the kernel pattern-completing across a real waking experience, with the speech pathway producing through tied weights, with the epistemic monitor governing the production register, and with a real NREM-REM cycle consolidating the day's traces into the substrate's longer memory, the demonstration is complete. What it demonstrates is that the architecture of experience is buildable, that memory does not require a context window, and that the gap between what biology does and what engineering can do is smaller than it looked the day before.
